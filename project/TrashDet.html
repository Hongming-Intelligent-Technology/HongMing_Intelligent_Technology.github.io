  项目概览

本项目实现了基于DQN（深度Q网络）算法的移动机器人避障与路径规划系统，支持激光雷达、视觉传感器、IMU多源数据融合， 适配室内复杂场景的自主导航需求。相比传统PID控制，强化学习模型在动态障碍物场景下的路径规划效率提升30%以上， 已在ROS2平台完成验证，可快速集成到差速底盘、履带式机器人等硬件载体。

核心特性：

  • 端到端强化学习训练框架，支持自定义奖励函数
  • 轻量化模型设计，适配边缘计算设备（如Jetson Nano）
  • ROS2接口封装，兼容主流机器人硬件生态
  • 可视化调试工具，实时监控机器人状态与决策过程
  技术架构

整体架构分层：

  1. 感知层：激光雷达数据预处理、视觉目标检测、IMU数据融合（基于卡尔曼滤波）
  2. 决策层：DQN强化学习模型（输入：环境状态；输出：运动指令）
  3. 执行层：ROS2控制节点、电机驱动接口、运动学逆解算
  4. 调试层：RViz可视化、TensorBoard训练监控、日志分析工具

核心依赖：

Python 3.8+、PyTorch 2.0、ROS2 Humble、OpenCV 4.8、NumPy 1.24
  快速开始

1. 环境安装

# 克隆仓库
git clone https://github.com/Hongming-Intelligent-Technology/Project-1.git
cd Project-1

# 创建虚拟环境
conda create -n robot-env python=3.8
conda activate robot-env

# 安装依赖
pip install -r requirements.txt
# 安装ROS2依赖（需提前配置ROS2环境）
sudo apt install ros-humble-ros-base ros-humble-rviz2

2. 模型训练

# 启动训练（默认使用仿真环境）
python train_dqn.py --env indoor-sim --epochs 1000 --batch-size 64

# 查看训练日志
tensorboard --logdir ./logs

3. 部署运行

# 启动ROS2节点（仿真环境）
ros2 launch robot_nav sim_launch.py

# 启动ROS2节点（真实硬件）
ros2 launch robot_nav hardware_launch.py --ros-args -p robot_type:=diff_drive
